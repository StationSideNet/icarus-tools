# STATION SIDE // ICARUS TOOLS

A collection of tools for the ICARUS PvE survival game.


## Talent Calculator

The ICARUS Talent Calculator helps Prospectors design stronger builds with confidence. Browse talent
trees, understand progression paths, and fine-tune point allocation for player and creature setups.
